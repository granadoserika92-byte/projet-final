# Guide de remplacement des couleurs

## Couleurs du logo
- Turquoise/Teal principal: #3EADB2 (teal-600)  
- Bleu foncé (maison): #0D4A52 (teal-900)
- Jaune (soleil): #F7C542 (amber-400)

## Replacements à faire
- `from-indigo-600 to-purple-600` → `from-teal-600 to-cyan-600`
- `from-indigo-700 to-purple-700` → `from-teal-700 to-cyan-700`
- `from-indigo-500 to-purple-500` → `from-teal-500 to-cyan-500`
- `from-purple-500 to-pink-500` → `from-amber-500 to-yellow-500`
- `text-indigo-600` → `text-teal-600`
- `text-purple-600` → `text-cyan-600`
- `bg-indigo-` → `bg-teal-`
- `bg-purple-` → `bg-cyan-`
- `border-indigo-` → `border-teal-`
- `border-purple-` → `border-cyan-`
- `hover:text-indigo-` → `hover:text-teal-`
- `hover:bg-indigo-` → `hover:bg-teal-`
